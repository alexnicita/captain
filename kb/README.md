# Business Knowledge Base (KB)

Execution-first repository for building and operating high-performance product and consulting businesses.

## How to use this KB

1. Start with **`00-meta/kb-operating-rules.md`**.
2. Pick one domain folder based on your immediate priority (sales, delivery, strategy, pricing).
3. Use each document's:
   - **Operating model** (how to run it)
   - **Checklist** (what to do now)
   - **KPI set** (how to track performance)
4. Run on weekly cadence; document decisions in your own logs.

## Structure

- `00-meta/` — KB standards, update cadence, and synthesis method
- `01-elite-product-consultancy/` — positioning, offer design, growth model
- `02-consulting-delivery-systems/` — delivery operations, quality, utilization, staffing
- `03-startup-strategy/` — market selection, GTM, strategic sequencing
- `04-product-studio-model/` — studio economics and portfolio operations
- `05-entrepreneurship/` — founder operating system and decision hygiene
- `06-pricing-profit-optimization/` — pricing architecture and margin mechanics
- `07-playbooks-templates/` — reusable checklists and meeting templates

## How to use this PDF library

- Start with `sources/pdf-library.md` and pick the category tied to your current objective.
- Use `sources/pdf-library-download-log.md` to open local copies in `assets/pdfs/` for offline reading and annotation.
- For execution speed, extract 3–5 actionable frameworks/checklists per PDF into the matching KB folder.
- Re-validate links quarterly and replace stale sources with official/open-access equivalents.

## Core principles

- **Outcomes over activity**: optimize for shipped business impact, not output volume.
- **Systemize before scaling**: build repeatable delivery and sales loops before hiring heavily.
- **Protect margins early**: weak pricing discipline compounds into fragile operations.
- **Short feedback loops**: weekly operating reviews, monthly strategy checks, quarterly resets.
