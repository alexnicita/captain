# Startup Go-to-Market System

## When to use this

Use when building first repeatable revenue motion or reworking a stalled GTM.

## Operating model

1. **ICP + trigger definition**
   - Define who buys, why they buy now, and what event triggers urgency.
2. **Message architecture**
   - Problem framing, differentiated promise, evidence, risk reversal.
3. **Channel portfolio design**
   - Pick 1 primary channel + 1 secondary test channel.
4. **Funnel instrumentation**
   - Track each stage from attention to retention.
5. **Weekly GTM review**
   - Diagnose by bottleneck: traffic, conversion, onboarding, retention, sales cycle.

## Checklist

- Single-source-of-truth pipeline dashboard
- Standard qualification framework (e.g., MEDDICC/BANT adapted)
- Onboarding flow mapped to first value moment
- Sales objections and response library maintained
- Win/loss interviews scheduled monthly

## KPIs

- MQL/SQL quality ratio
- Demo-to-close conversion
- Sales cycle length
- Activation rate within 7/14 days
- CAC payback proxy and gross margin contribution

## Failure modes

- Too many channels before one channel works
- No shared language between marketing, product, and sales
- Onboarding neglected while top-of-funnel spending increases
- No clear owner for GTM system performance
